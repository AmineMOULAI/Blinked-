# Blinked 💡

> La plateforme simple pour permettre à vos équipes d'échanger conseils, bonnes pratiques et retours d'expérience au quotidien.

![Aperçu du projet](public/images/logo_1.png) 
## 📖 À propos

Aujourd'hui, dans de nombreuses entreprises, les connaissances restent bloquées dans des boucles d'e-mails infinies ou se perdent après des réunions non documentées. Résultat : une perte de temps et une frustration au quotidien pour les équipes.

**Blinked** a été conçu pour résoudre ce problème. C'est un outil collaboratif moderne, sécurisé et intuitif qui transforme l'expérience collaborateur. Notre plateforme centralise le savoir interne et valorise l'expertise de chaque membre de l'équipe, permettant ainsi un gain de temps mesurable et une augmentation significative de la productivité.

## ✨ Fonctionnalités Principales

- **💬 Fil de conseils** : Publiez et consultez des conseils en temps réel.
- **🔍 Recherche intelligente** : Trouvez rapidement les réponses et solutions déjà partagées.
- **👤 Profils experts** : Identifiez facilement les personnes ressources au sein de votre entreprise.
- **🔔 Notifications ciblées** : Recevez uniquement les contenus pertinents pour votre domaine.
- **📊 Impact mesurable** : Suivi des gains de temps et de l'augmentation de la productivité.

## 🛠️ Technologies Utilisées

Ce projet a été développé avec des technologies web modernes pour garantir des performances optimales et une expérience utilisateur fluide :

- **Framework** : [Next.js](https://nextjs.org/) (App Router)
- **Bibliothèque UI** : [React](https://reactjs.org/)
- **Style** : [Tailwind CSS](https://tailwindcss.com/)
- **Animations** : [Framer Motion](https://www.framer.com/motion/)
- **Composants** : [Shadcn UI](https://ui.shadcn.com/) & Radix UI
- **Icônes** : [Lucide React](https://lucide.dev/)
- **Langage** : TypeScript

## 🚀 Installation et Lancement en local

Si vous souhaitez faire tourner ce projet sur votre machine locale, suivez ces étapes :

### Prérequis
Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé sur votre machine.

### Instructions

1. **Cloner le dépôt**
   ```bash
   git clone [https://github.com/votre-pseudo/Blinked.git](https://github.com/votre-pseudo/Blinked.git)
   cd Blinked

2.  **Installer les dépendances**

    ```bash
    npm install
    ```

3.  **Lancer le serveur de développement**

    ```bash
    npm run dev
    ```

4.  **Ouvrir l'application**
    Ouvrez [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) dans votre navigateur pour voir le résultat.

## 📦 Scripts disponibles

  - `npm run dev` : Lance l'environnement de développement.
  - `npm run build` : Compile l'application pour la production.
  - `npm run start` : Lance l'application compilée en mode production.
  - `npm run lint` : Analyse le code pour trouver d'éventuelles erreurs (ESLint).
