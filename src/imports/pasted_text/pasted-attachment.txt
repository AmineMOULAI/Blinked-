💻 3. Contraintes Techniques (Next.js)

Génère la structure du code en Next.js (App Router ou Pages Router).

Utilise Tailwind CSS pour le styling, car cela permet un code propre, lisible et un responsive design facile (mobile-first obligatoire).

Utilise Framer Motion pour gérer les animations fluides au scroll.

Divise le code en petits composants réutilisables (ex: Hero.jsx, FeatureCard.jsx, Testimonial.jsx) pour respecter la consigne du "Code propre et structuré".

Donne-moi le code complet de la page principale (page.js ou index.js) en important ces composants avec des placeholders pour les images (/images/illustration1.jpg, etc.).
